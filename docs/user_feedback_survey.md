# User Feedback Survey Template

Use this after showing participants the top 5 recommendations for one learner profile. A small sample of 5-10 participants is enough for prototype feedback if clearly described as formative evaluation.

## Participant Context

1. Which pathway did you review?
   - Data Analyst
   - Machine Learning Engineer
   - Software Developer

2. How familiar are you with this pathway?
   - Not familiar
   - Slightly familiar
   - Moderately familiar
   - Very familiar

## Recommendation Usefulness

Rate each statement from 1 to 5:

| Score | Meaning |
|---|---|
| 1 | Strongly disagree |
| 2 | Disagree |
| 3 | Neutral |
| 4 | Agree |
| 5 | Strongly agree |

Questions:

1. The recommended resources are relevant to the target career pathway.
2. The recommended resources match the learner's weak skills.
3. The recommended resources are suitable for the learner's current level.
4. The top-ranked resource is a good first recommendation.
5. I would find this recommendation list useful if I were learning this pathway.

## Explanation Clarity

Rate each statement from 1 to 5:

1. The explanation makes it clear why each resource was recommended.
2. The explanation connects the recommendation to skill gaps.
3. The explanation connects the recommendation to the career pathway.
4. The explanation would help a learner decide what to study next.

## Open Feedback

1. Which recommendation was most useful and why?
2. Which recommendation seemed least useful and why?
3. What extra information would make the recommendation more trustworthy?
4. Are there any important skills or resources missing from the list?

## Suggested Analysis

Calculate the mean score for:

- pathway relevance
- skill-gap match
- difficulty suitability
- explanation clarity
- overall usefulness

Then include a short qualitative summary of common comments.
