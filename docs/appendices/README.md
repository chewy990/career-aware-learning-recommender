# Appendices

This folder keeps the supporting material for the Appendices section of the Preliminary Project Report.

Use these files as organised source material when preparing the final submitted report. The appendix content should be cross-referenced from the main report text before submission.

| Appendix | File | Purpose |
|---|---|---|
| Appendix A | `appendix_a_dataset_field_summary.md` | Summarises the project datasets and important fields |
| Appendix B | `appendix_b_skill_map_method.md` | Explains the 0-3 pathway skill map and evidence approach |
| Appendix C | `appendix_c_evaluation_metric_formulas.md` | Defines Precision@K, Recall@K, and NDCG@K |
| Appendix D | `appendix_d_prototype_screenshots.md` | Placeholder list for screenshots to add from the Streamlit app |

Do not paste full raw CSV files into the report unless specifically required. Summaries are usually clearer and more appropriate for appendices.
